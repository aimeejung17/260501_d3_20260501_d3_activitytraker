# STATUS — 중단 메모

이 저장소는 FlowXP(게임형 SNS 디톡스) **MVP 데모**를 만들기 위한 프로젝트였고, 현재는 **실기기 테스트 단계에서 중단**된 상태야.

## 지금 되는 것

- **웹 데모**: `docs/index.html`
  - UI/규칙(점수→XP→레벨) “느낌” 공유용
  - 실제 SNS 사용시간과는 연동 안 됨

## 지금 안 되는 것(아직 못한 것)

- **안드로이드 실제 테스트**
  - `UsageStats` 권한/데이터 확인
  - 폰에서 실제 사용시간 수집값 변화 확인

## 중단 이유

- Windows PC에서 **Android Studio 설치 중 용량 부족**으로 중단
- 설치 흔적은 삭제 완료(Program Files 쪽 제거)

## 다음에 다시 할 때 할 일 (최단 루트)

1. C: 여유공간 확보 (권장 15~20GB 이상)
2. Android Studio 설치
3. 프로젝트 열기 → Gradle Sync
4. **Build APK(s)**로 APK 만들기 또는 폰 연결해서 Run
5. 폰에서 “사용 정보 접근” 권한 허용 후 실제 값 확인
